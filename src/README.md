Source
===

MEIMEIを構成するソースコードです。

C/Assemblyで書かれたコードと、成果物を作成するためのリンカスクリプトが存在します。


## ディレクトリ構成

ソースコードはそれぞれの役割ごとに分けています。

name | role
--- | ---
boot_loader | 様々な情報をメモリ上に展開します。
kernel | マシンの資源管理を行います。
libc | Cの標準ライブラリの一部が実装されています。
meimei | アプリケーション本体です。

```
+-------------+   +--------+   +------+   +--------+
| boot_loader +--->        <---+ libc <---+        |
+-------------+   | kernel |   +------+   | meimei |
                  |        <-------------->        |
                  +--------+              +--------+
```
